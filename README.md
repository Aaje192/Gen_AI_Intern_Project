# Gen_AI_Intern_Project

# Cam Ring Edge Detection and Dimensional Analysis

## Objective
To detect the inner and outer edges of a cam ring from a top-view image and measure
the radial distance between them at regular angular intervals.

## Methodology
- Image preprocessing and Canny edge detection
- Contour extraction
- Center estimation
- Conversion to polar coordinates
- Radial distance computation

## Tools Used
- Python
- OpenCV
- NumPy
- Matplotlib
- VS Code

## Output
- Processed image with detected edges
- CSV file containing distance vs angle data
